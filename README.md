# Hostel Grievance Portal

A web application designed to streamline the process of reporting and managing grievances in a hostel environment. Detailed for reporting issues related to food, room maintenance, Wi-Fi, and more.

## Features
- **User-Friendly Interface**: Simple and intuitive design for easy navigation.
- **Categorized Reporting**: Specific sections for different types of grievances.
- **Direct Reporting**: Streamlined process to submit issues directly to administration.

## Prerequisites
- Python 3.x
- pip (Python package installer)

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/amark2005/Hostel-Grievance-Portal.git
   cd HostelPortal
   ```

2. **Create and activate a virtual environment (optional but recommended):**
   ```bash
   python -m venv venv
   # On Windows
   venv\Scripts\activate
   # On macOS/Linux
   source venv/bin/activate
   ```

3. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

## Usage

1. **Run the application:**
   ```bash
   python app.py
   ```

2. **Access the portal:**
   Open your web browser and navigate to `http://localhost:5000` (or the address shown in the terminal).

## Technologies Used
- **Backend**: Flask (Python)
- **Frontend**: HTML, CSS
